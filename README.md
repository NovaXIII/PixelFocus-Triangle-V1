# PixelFocus Triangle V1

![Preview](https://cdn.modrinth.com/data/cached_images/0f47bd438e793c8ca4f30f164b7035e78b8b1b29.png)

## Overview
PixelFocus Triangle V1 is a precision-focused Minecraft resource pack designed to replace the default crosshair with a clean triangular aiming indicator. Built for players who value minimal UI and maximum visual clarity, this pack enhances aim stability without distracting elements.

## Features
- Sharp triangular crosshair design optimized for center-screen focus
- Minimal visual interference for PvP and survival gameplay
- Lightweight implementation with zero performance impact
- Compatible with most shaders and GUI overhaul packs
- Works across multiple Minecraft versions via adaptive texture layering

## Design Philosophy
The triangle shape was chosen to naturally guide the eye toward directional alignment. Unlike traditional crosshairs, it reduces visual noise while still providing clear positional feedback, making it ideal for competitive environments.

## Compatibility
Supports a wide range of Minecraft Java Edition versions depending on pack configuration, including modern 1.20+ builds with overlay support.

## Use Case
Best suited for PvP players, speedrunners, and cinematic gameplay creators who require a clean and distraction-free HUD.
